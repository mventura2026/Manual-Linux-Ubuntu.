Introducción

Linux es un sistema operativo de código abierto creado en 1991 por Linus Torvalds. A diferencia de Windows, Linux es gratuito, seguro, altamente personalizable y utilizado en todo el mundo: desde teléfonos Android hasta los servidores que alimentan internet.

Ubuntu es la distribución de Linux más popular para principiantes, con interfaz amigable, enorme comunidad de soporte y miles de programas gratuitos. Ubuntu 24.04 LTS recibe soporte oficial hasta 2029.

CaracterísticaLinux UbuntuWindows 11
CostoGratuitoLicencia de pago (~$139)
SeguridadMuy alta (pocos virus)Moderada (requiere antivirus)
Código fuenteAbierto (modificable)Cerrado (propietario)
RendimientoExcelente en hardware antiguoRequiere hardware reciente
PrivacidadAlta (sin telemetría forzada)Moderada (recopila datos)
✅
DATO IMPORTANTEUbuntu funciona perfectamente en computadoras donde Windows 11 ya no es compatible. ¡Es una excelente opción para dar nueva vida a equipos antiguos!

02

Descarga del ISO Oficial de Ubuntu

El archivo ISO contiene el sistema operativo completo. Descárgalo únicamente desde el sitio oficial.

  1. Abre tu navegador y ve a ubuntu.com/download/desktop
  2. Haz clic en el botón Download 24.04 LTS.
  3. El archivo se llama ubuntu-24.04-desktop-amd64.iso (~5.7 GB).
  4. Espera a que la descarga se complete (15–45 minutos según tu conexión).
📷Página oficial de Ubuntu mostrando el botón de descarga de la versión 24.04 LTS
⚠️
IMPORTANTEDescarga SOLO desde ubuntu.com. Sitios no oficiales pueden ofrecer versiones modificadas que contienen malware.
03

Creación de la USB Booteable

⚠️
ADVERTENCIALa USB perderá TODOS sus datos durante este proceso. Guarda tus archivos importantes antes de continuar.

Opción A: Usando Rufus (Windows)

  1. Descarga Rufus desde rufus.ie (gratuito, no requiere instalación).
  2. Conecta la USB y abre Rufus.
  3. En Dispositivo, selecciona tu USB.
  4. En Elección de arranque, selecciona el ISO de Ubuntu.
  5. Esquema: GPT · Sistema de destino: UEFI (no CSM).
  6. Haz clic en EMPEZAR y espera a que diga LISTO.
📷Rufus configurado con el ISO de Ubuntu, esquema GPT y destino UEFI

Opción B: Usando Balena Etcher (más sencillo)

  1. Descarga desde balena.io/etcher e instálalo.
  2. Haz clic en Flash from file y selecciona el ISO de Ubuntu.
  3. Haz clic en Select target y selecciona tu USB.
  4. Haz clic en Flash! y espera (10–15 minutos).
💡
CONSEJOBalena Etcher es más sencillo para principiantes. Rufus ofrece más opciones avanzadas. Ambas funcionan perfectamente.
04

Configuración de la BIOS/UEFI

Para que la computadora arranque desde la USB, necesitamos entrar a la BIOS y cambiar el orden de arranque.

Marca del equipoTecla BIOSBoot Menu rápido
ASUSF2 o Supr (Del)F8
MSISupr (Del)F11
GigabyteSupr (Del) o F2F12
HPF10 o EscF9
DellF2F12
LenovoF1, F2 o Fn+F2F12
AcerF2 o Supr (Del)F12

Cambiar el Boot Order

  1. Ingresa a la BIOS presionando la tecla correspondiente al encender.
  2. Ve a la sección Boot → Boot Priority.
  3. Mueve la USB al primer lugar con las teclas + / - o arrastrando.
  4. Presiona F10 para guardar y salir.
📷BIOS mostrando el Boot Order con la USB en primer lugar
💡
CONSEJOUsa el Boot Menu rápido (F12, F11 o F9) para seleccionar la USB directamente sin cambiar la configuración permanente de la BIOS.
05

Arranque desde USB

  1. Conecta la USB booteable al computador.
  2. Apaga y vuelve a encender el equipo.
  3. Aparecerá el menú GRUB de Ubuntu.
  4. Selecciona Try or Install Ubuntu y presiona Enter.
📷Menú GRUB de Ubuntu mostrando la opción "Try or Install Ubuntu"
🖥️
MODO LIVEUbuntu te permite probar el sistema ANTES de instalarlo, sin modificar tu disco duro. Selecciona Try Ubuntu para explorar. Cuando estés listo, haz clic en el ícono Install Ubuntu del escritorio.

07

Configuración Inicial del Sistema

Al iniciar Ubuntu por primera vez, sigue el asistente de bienvenida y luego actualiza el sistema.

Actualizar el sistema (muy importante)

Abre la Terminal con Ctrl + Alt + T y ejecuta:

sudo apt update
sudo apt upgrade -y
sudo apt autoremove -y
💡
CONSEJOsudo ejecuta el comando como administrador. Te pedirá tu contraseña. Al escribirla no verás nada en pantalla — es normal, es por seguridad.

Instalar apps esenciales

# Instalar VLC (reproductor de video)
sudo apt install vlc -y

# Instalar GIMP (editor de imágenes)
sudo apt install gimp -y

# Instalar Git
sudo apt install git -y

Solución de Problemas Comunes

01La USB no arranca / el equipo la ignora

Causa: Boot Order incorrecto o Secure Boot bloqueando el arranque.

  • Usa el Boot Menu rápido (F12/F11/F9) y selecciona la USB directamente.
  • En la BIOS, desactiva temporalmente Secure Boot.
  • Recrea la USB con Rufus usando esquema GPT y sistema UEFI.
  • Prueba en un puerto USB diferente (preferiblemente USB 2.0).
02Pantalla negra al iniciar Ubuntu

Causa: Driver de tarjeta gráfica incompatible (frecuente con Nvidia).

  • En el menú GRUB, selecciona Ubuntu (safe graphics).
  • Una vez instalado: Configuración → Software y actualizaciones → Controladores adicionales.
  • Desde terminal: sudo ubuntu-drivers autoinstall
03Sin conexión Wi-Fi

Causa: Driver de tarjeta inalámbrica no instalado.

  • Conéctate temporalmente con cable Ethernet.
  • Ejecuta: sudo ubuntu-drivers autoinstall
  • Reinicia: sudo reboot
04Ubuntu no aparece al arrancar (Dual Boot)

Causa: Windows tomó control del arranque o GRUB no se instaló.

  • Arranca desde la USB en modo prueba (Try Ubuntu).
  • Ejecuta: sudo grub-install /dev/sda
  • Luego: sudo update-grub y reinicia sin la USB.
05La instalación se congela

Causa: USB defectuosa, RAM con errores o disco dañado.

  • Espera al menos 30 minutos antes de asumir que está congelado.
  • Recrea la USB con una memoria diferente.
  • En el menú GRUB selecciona "Check disk for defects" para verificar el ISO.
10

Recomendaciones Finales

  • Actualiza semanalmente: sudo apt update && sudo apt upgrade -y
  • Haz copias de seguridad con Déjà Dup (incluido en Ubuntu) o guarda archivos en la nube.
  • Explora Ubuntu Software Center para encontrar alternativas gratuitas a programas de pago.
  • Aprende más en: ubuntu.com/tutorials · linuxcommand.org · askubuntu.com
  • Equipos lentos: Prueba Xubuntu o Lubuntu, versiones más ligeras de Ubuntu.
